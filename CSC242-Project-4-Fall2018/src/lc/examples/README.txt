
Test drivers for the PerceptronClassifier and LogisticClassifier implementations. You might want to do different things to show off your program or gather statistics for your writeup.

The earthquake-clean and earthquake-noisy datasets are from AIMA, so you can duplicate those experiments.

Other datasets from UCI Machine Learning Repository (http://archive.ics.uci.edu/ml/index.php) for you to test your decision tree learner on. The ".names" file explains the data in the ".data" file.

- The Congressional Voting Records (``house votes 84'') dataset
  http://archive.ics.uci.edu/ml/datasets/Congressional+Voting+Records
  - modified to use numerical values for the attributes and class label; see .names file for explanation
